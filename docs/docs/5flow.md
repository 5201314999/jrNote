# flow 简单介绍

## 1 介绍
flow 是 facebook 出品的 JavaScript 静态类型检查工具,类似java 函数中传参声明类，函数返回时亦声明返回类型。

```
// @flow
function add(a:number, b: number): number {
    return a + b;
}

//会报错
add('1'+2);

```
