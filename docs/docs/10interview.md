## 网络知识

1. TCP/IP 协议

TCP/IP 协议是一个协议族，主要是定义网络中2台计算机的通信规则。分为四层

* 应用层 （ftp,http)
* 传输层 (tcp,udp)
* 网络层 (数据包，ip 协议)
* 数据链路层 (硬件部分，网卡)

2. 专业术语
   ARP 协议 Address Resolution Protocol
   MAC Media Access Control Address

3. TCP 3次握手（确保双方可以正常通信） ,TCP 4 次挥手（断开连接）

具体步骤内容

发生时间点：
3次握手：其中2个标志位 SYN 和 ACK；
http 请求和响应。
4次挥手：其中2个标志位 FIN 和 ACK；
